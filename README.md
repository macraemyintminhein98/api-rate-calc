# API Rate Limit Calculator

This simple web application helps developers understand and configure API rate limits. Input parameters like requests per second/minute/hour, burst limits, and concurrency, and get actionable outputs on recommended delays and maximum requests over various periods.

## Features
- Calculates recommended delay between API calls.
- Shows maximum requests per second, minute, and hour.
- Accounts for burst limits and maximum concurrent requests.

## How to Use
1. Enter your API's rate limit parameters in the provided fields.
2. Click 'Calculate Rate Limits' to see the results.
3. (Optional) Unlock advanced features with a one-time purchase.

## Development
Built with basic HTML, CSS, and JavaScript. The calculation logic is client-side.
